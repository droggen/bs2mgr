Version

Purpose

Bs2Mgr is a management tool for BlueSense sensor nodes. It provides:
  • Node control: rapid access to common node control functions (e.g. synchronisation, starting motion mode, etc)
  • Annotations: open-loop annotation (i.e. without connection to the node), once nodes are synchronized

Connectivity

BlueSense nodes can be interfaced over serial ports (USB or hardware serial) or Bluetooth RFCOMM.

Annotations

The annotation tab allows to perform open-loop annotations after the nodes have been synchronized to the system time. BlueSense timestamps indicate the number of milliseconds elapsed since the first day of the month.

Limitations

Bluetooth is not supported under Windows.
